# Web Development Practice

A collection of practice layouts, web designs, and interactive calculators built using HTML5 and vanilla CSS.

## Projects

- **`Lab-1/`**
  - An interactive GPA and marks aggregate calculator designed with clean CSS styles. Useful for students to calculate aggregate scores based on grading criteria.

- **`Lab-2/`**
  - A responsive homepage layout utilizing semantic HTML elements and custom CSS stylesheets.

- **`Lab-3/`**
  - A multi-page mock website implementation containing:
    - `index.html` — Homepage
    - `about.html` — About Us page
    - `product.html` — Product showcase page
    - `checkout.html` — Order checkout form
    - `login.html` & `signup.html` — User authentication screens
    - `style.css` — Standard styles for all pages

## How to Run

Simply open any of the `.html` files in your web browser, or run a local development server:
```bash
# Using VS Code Live Server extension, or simple Python server:
python -m http.server 8000
```
Then navigate to `http://localhost:8000` in your web browser.
